# 📚 Vision Communication & Bookshop Management System

A complete **Java Desktop Application** developed for automating bookshop operations including billing, stock management, GRN handling, and profit reporting.

🎓 Developed as part of *Software Application Design – Final Viva Examination*

## 🛠️ Technologies Used

- Java (JDK 8)
- Java Swing (GUI)
- MySQL Database
- Windows Environment

---

## 🚀 Features

### 🔐 Authentication
- Secure Login System
- Change Login Credentials (Security Code Protected)

### 🧾 Billing Module
- Bill Products
- Apply Quick Discounts
- Generate Customer Bill Report
- Prevent under-value payments

### 📦 GRN (Goods Received Note)
- Add New Stock
- Manage Buyer Payments
- Generate GRN Report
- Support “Not Today Paying” option

### 📊 Inventory Management
- Add / Edit / Delete Products
- Manage Brands & Categories
- Update Selling Prices
- Search & Filter Stock

### 📈 Reports & Analytics
- Stock Report
- Daily Profit Report
- Monthly Profit Report
- Total Profit Report
- Total Sales Income View

### 💾 System Utilities
- Instant Database Backup
- Theme Settings
- System Configuration

---

## 🖥️ System Requirements

### Hardware
- Minimum 2GB RAM
- 5GB Free Disk Space
- 1024x768 Display
- Printer (for reports)

### Software
- Windows 7 or above
- Java JDK 8
- MySQL Server

---

## 🗄️ Database Information

The system stores:

- Product Name
- Buying Price
- Selling Price
- Quantity
- Brand & Category
- Billing Date
- Buyer Information
- Payment Details
- Profit Records
- Product Expiry Date (if available)
